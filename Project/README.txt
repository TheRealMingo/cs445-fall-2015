(1) Run the start.sh file with the command: "bash start.sh"
(2) If this files download dos2unix by running: "apt-get install dos2unix"
(3) Run "dos2unix start.sh"  
(4) Now run the command "bash start.sh" again
(5) Open web browser and go to "localhost:8080/system.SAD"
(6) Have Fun! 
(7) There is a link the api on the homepage

Note urls are in the following format due to the natural of Tomcat on Ubuntu

localhost:8080/system.SAD/sad/...

Also due to the natural of Tomcat 
the only invalid url is 

localhost:808/system.SAD/sad
