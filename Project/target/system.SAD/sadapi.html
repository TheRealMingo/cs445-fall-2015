POST /advertiser

Creates an advertiser

Parameters

  • fname(required) : The first name of advertiser
  • mname(optional) : The middle name of advertiser
  • lname(required) : The last name of advertiser
  • suffix(optional) : The suffix of the name of the advertiser
  • emailAddr (required): The advertiser's email address
  • businessName (required): The advertiser's business's name
  • phoneNumber(required): The advertiser's phone number
  • startDate(required) : The start date of advertiser - format: MMM dd, YYYY
  • socialMedia (optional) : An array of social media accounts of the advertiser - format : "{outlet}:{content}"

Resource Url

/advertisers/advertiser

Example Request

GET http://localhost:8080/sad/advertisers/listings/advertiser

PUT /advertiser/{id}

POST /listing

Creates a listing

Parameters

  • Advertiser(required) : The advertiser for the listing
  • businessDescr(required) : A description of the business for the listing
  • imageLoc(optional) : An image for the listing
  • websiteLink(optional) : The website of the business that own the listing
  • startDate(required): The date the listing will be to be active
  • endDate(required): The date the listing will no longer be active
  • price(required): The price of the listing

Resource Url

/listings/listing

Example Request

GET http://localhost:8080/sad/listings/listing

JSON Formats

Category Format

Category without subcategory

			{
				"categoryGenre":"{category}",
				"subcat": []
			}

Category with subcategory

			{
				"categoryGenre":"{category}",
				"subcat":[
					{
						"categoryGenre":"{category}", 
						"subcat":[]
					}
				]
			}

Listing Format

			{
				"advertiser":[{
					"fname":"{firstname}",
					"mname":"{middlename}", 
					"lname":"{lastname}", 
					"suffix":"{suffix}", 
					"emailAddr":"{emailAddr}", 
					"businessName":"{businessName}", 
					"phoneNumber":"{phoneNumber}", 
					"startDate":"{startDate}",
					"socialMedia": "{socialMedia}"
				}],
				"businessDescription":"{description}", 
				"imageLoc":"{imageLoc}", 
				"websiteLink":"{websiteLink}", 
				"startDate":"{startDate}", 
				"endDate":"{endDate}", 
				"price":JSON_NUMBER
			}

Advertiser Format

			{
				"fname":"{firstname}",
				"mname":"{middlename}", 
				"lname":"{lastname}", 
				"suffix":"{suffix}", 
				"emailAddr":"{emailAddr}", 
				"businessName":"{businessName}", 
				"phoneNumber":"{phoneNumber}", 
				"startDate":"{startDate}",
				"socialMedia": "{socialMedia}"
			}
